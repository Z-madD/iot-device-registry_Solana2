# IoT Device Registry on Solana

Backend descentralizado para registrar, leer, actualizar y eliminar dispositivos IoT utilizando Solana y Anchor Framework.

## 🚀 Tecnologías

- **Rust** + **Anchor** (v0.30.1)
- **Solana Devnet**
- Cuentas con **PDA** (Program Derived Address)
- TypeScript para pruebas
